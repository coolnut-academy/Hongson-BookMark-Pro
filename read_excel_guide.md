ไฟล์นี้คือ Excel เก่าแบบ .xls (Excel 97-2003), ไม่ใช่ .xlsx

สำคัญมาก:
- ห้าม assume ว่า pandas.read_excel() จะอ่านได้ตรง ๆ
- ห้ามรันแบบเงียบแล้วค้าง
- ต้องใช้แนวทาง defensive parsing สำหรับ legacy Excel

ให้ทำงานตามลำดับนี้:

1) ตรวจสอบไฟล์ก่อน
- เช็กว่าไฟล์มีอยู่จริง
- แสดง path
- แสดงขนาดไฟล์
- ยืนยันว่าเป็น .xls

2) Inspect workbook ก่อน
- อย่าเพิ่งโหลดทั้งไฟล์ทันที
- ใช้วิธีที่เหมาะกับ Excel 97-2003
- ดึงชื่อ sheet ทั้งหมดออกมาก่อน
- แสดงจำนวน sheet

3) Preview แบบปลอดภัย
- อ่านเฉพาะ sheet แรกก่อน
- อ่านแค่ 20 แถวแรก
- แสดง:
  - shape
  - columns
  - head()
  - non-null count ต่อคอลัมน์

4) วิเคราะห์โครงสร้าง
- ตรวจว่า header จริงเริ่มแถวไหน
- มีแถวหัวกระดาษ / merged title / ข้อมูลเกินด้านบนหรือไม่
- มี unnamed columns หรือไม่
- เสนอว่าควรใช้ header=0 หรือ skiprows เท่าไร

5) เสนอ parsing strategy
- ควรใช้ engine อะไร
- ควรอ่าน sheet ไหน
- ควรใช้ header/skiprows/usecols อย่างไร
- ควร clean column names อย่างไร

6) ค่อยอ่านจริง
- ถ้าขั้นไหน fail ให้แสดง error เต็ม ๆ
- ถ้าต้อง fallback ให้ fallback เอง
- ห้ามหยุดเงียบ

ข้อกำหนด:
- log ทุกขั้นแบบ [1/6], [2/6], ...
- เขียน Python code ที่รันได้จริงบน Windows
- รองรับ path ภาษาไทยและช่องว่าง
- อย่า hardcode เฉพาะไฟล์เดียว เพราะไฟล์นี้เป็น input ของระบบงานทะเบียน/วัดผลในโรงเรียน
